# Azure AI Chat

Azure AI Chatは、文教大学の学生向けに作られたローカル・ウェブサイトです。友人とのリアルタイムチャット機能に加え、Azure AIを活用した大学専用のAIアシスタント機能を搭載しています。

## ✨ 主な機能 (Features)

### 1. フレンドチャット (Friend Chat)
* **概要:** 大学の友人たちと気軽にメッセージのやり取りができる専用のチャットボックスです。
* **特徴:** 学生同士のコミュニケーションを円滑にするためのシンプルで使いやすいUIを提供します。

### 2. Azure AIチャットアシスタント (Azure AI Chat Box)
* **概要:** Azure AIをバックエンドに採用した、文教大学専用のインテリジェントな質問応答システムです。
* **大学特化型AI:** システム内のローカルデータを読み込み、ユーザーからの質問に回答します。
* **情報のフィルタリング:** AIは「文教大学に関する質問」にのみ回答するように設定されています。大学に関係のない質問には応答しないため、情報が脱線することなく、正確で関連性の高いサポートを提供します。

# Azure AI Chat

**Azure AI Chat** is a local web platform built specifically for Bunkyo University students. It combines a real-time messaging system for friends with an intelligent, university-specific AI assistant powered by Azure AI.

## ✨ Features

### 1. Friend Chat Box
* **Overview:** A dedicated chat interface allowing you to connect and message your university friends in real-time.
* **Benefits:** Provides a simple and accessible way for students to collaborate and socialize on a local platform.

### 2. Azure AI Chat Box
* **Overview:** A smart assistant powered by Microsoft Azure AI that serves as a dedicated knowledge base for the university.
* **System Data Integration:** The AI reads through local system data to provide accurate and relevant answers to user inquiries.
* **Strict University Focus:** The AI is specifically prompted to answer *only* questions related to Bunkyo University. If a user asks an unrelated question, the AI will politely decline, ensuring the chat remains a focused and reliable academic tool.
